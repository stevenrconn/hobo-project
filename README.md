# hobo-project
## Minimal Vagrant boxes for VirtualBox and Parallels desktop VMs

### Supported distros
#### Parallels (aarch64)
* Fedora 40
* Rockylinux 9
* Ubuntu 22.04 LTS "Jammy"
* Ubuntu 24..04 LTS "Noble"

#### VirtualBox (x86_64)
* Debian 11 "Bullseye"
* Debian 12 "Bookworm"
* Fedora 40
* RockyLinux 8
* RockyLinux 9
* Ubuntu 22.04 "Jammy"
* Ubuntu 24.04 "Noble"
